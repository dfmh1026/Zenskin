# Zen Skin Studio

Landing page para el centro de estética Zen Skin Studio.

## Despliegue en GitHub Pages

Esta página está preparada para publicarse en GitHub Pages desde la carpeta raíz del repositorio.

## Archivos importantes

- index.html: estructura principal de la landing page
- styles.css: estilos visuales
- responsive.css: estilos responsive
- script.js: comportamiento del slider y formulario
- .nojekyll: evita que GitHub Pages procese el sitio con Jekyll
